# Senfelds_pica :pizza:

**Java picas pasūtīšanas aplikācija**, kur lietotājam ir iespēja **izveidot profilu**, **piereģistrēties** vai **pieslēgties profilam**. 

Lietotājs var **izveidot**, **pasūtīt** uz mājām vai uz vietas un **dzēst izveidoto picu**.
